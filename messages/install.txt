SublimeLinter-contrib-lslint
-------------------------------

This linter plugin for SublimeLinter provides an interface to lslint.

** IMPORTANT! **

* lslint binaries are not part of this package, however they are
  required to be in your **PATH**.
* language grammar, highlighting, completions and tooltips are not
  part of this package.

Before this plugin will activate, you *must*
follow the installation instructions here:

https://github.com/xenhat/SublimeLinter-contrib-lslint
