SublimeLinter-contrib-lslint
-------------------------------

This linter plugin for SublimeLinter provides an interface to lslint.

** IMPORTANT! **

* Having SublimeLinter 4 installed is NOT RECOMMENDED, and UNTESTED
* lslint binaries are not part of this package, however they are
  required to be present in the **LSL package's proper folder**, or in your **PATH**.
* language grammar, highlighting, completions and tooltips are not
  part of this package.

Before this plugin will activate, you *must* follow the installation instructions here:

https://github.com/xenhat/SublimeLinter-contrib-lslint

The short version is:

* Uninstall Sublime Linter 4 if applicable
* Add the following repository: https://github.com/SublimeLinter/SublimeLinter/tree/v3
* Install Sublime Linter
* Install lslint somewhere in your path
